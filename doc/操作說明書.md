## 系統概覽

### 登入與班別選擇
1. 開啟系統後會顯示**班別選擇視窗**
2. 選擇您的班別（A班/B班/C班）或點選對應操作員
3. 點擊「**選取**」進入系統

> ⚠️ **注意**：班別資訊會記錄在生產報表中，請確認選擇正確

### 主導航列
系統頂部有 6 個主要功能按鈕：
- 即時監控 (Monitor) - 預設首頁
- 生產排程 (Schedule)
- 生產報表 (Report)
- 生產分析 (Analysis)
- 保養維修 (Maintenance)
- 系統設定 (Settings)

## 1. 即時監控 (Monitor)

### 功能說明
即時監控頁面是操作員最常使用的畫面，用於監看當前生產狀態。

### 畫面區塊

| 區塊 | 說明 |
|------|------|
| **生產數量** | 顯示當前工單的累計生產張數 |
| **車速** | 顯示目前機台運行速度 (m/min) |
| **剩餘數量** | 目標數量 - 已生產數量 |
| **排程佇列** | 下方顯示待生產的工單清單 |

### 功能鍵操作

| 按鍵 | 功能 | 說明 |
|------|------|------|
| **F1** | 上移 | 將選取的工單往上移動一格 |
| **F2** | 下移 | 將選取的工單往下移動一格 |
| **F3** | 送單 | 開始生產當前工單（需先停機） |
| **F4** | 完工 | 結束當前工單（需先停機） |
| **F5** | 良品 +1 | 手動增加生產計數 |
| **F6** | 不良 -1 | 手動減少生產計數（不良扣除） |
| **F7** | 訂單 | 開啟訂單詳情視窗 |
| **F8** | 下一筆 | 切換自動/手動下一筆模式 |
| **F9** | 班別 | 重新選擇班別 |
| **F10** | 退回 | 將當前工單退回佇列（需先停機） |
| **F12** | 離開 | 退出系統 |

### 標準操作流程

```
1. 確認排程 → 2. 按 F3 送單 → 3. 開始生產 → 4. 生產完成按 F4 → 5. 確認完工數量
```

### 故障排除

| 問題 | 可能原因 | 解決方式 |
|------|----------|----------|
| 無法按 F3 送單 | PLC 連線中斷 | 檢查左下角連線狀態，重新連線 |
| 車速顯示 0 | 訊號中斷或機台未啟動 | 確認機台電源，檢查 PLC 連線 |
| 數量不跳動 | 光電感測器異常 | 通知維護人員檢查感測器 |
| F4 完工無反應 | 車速不為 0 | 需先停機（車速為 0）才能完工 |

## 2. 生產排程 (Schedule)

### 功能說明
管理生產工單順序與產品資料庫。

### 畫面區塊

| 區塊 | 說明 |
|------|------|
| **左側 - 排程佇列** | 顯示待生產的工單清單，可調整順序 |
| **左側 - 紙箱展開圖** | 選取工單後顯示對應的紙箱規格圖 |
| **右側 - 產品資料庫** | 產品規格資料，用於新增排程 |

### 操作步驟

#### 調整排程順序
1. 在左側選取要移動的工單
2. 點擊「**上移**」或「**下移**」按鈕
3. 點擊「**順序重置**」可重新編號（10, 20, 30...）

> ⚠️ **注意**：正在生產中的工單（第一筆）無法移動

#### 新增排程
1. 在右側產品資料庫選取產品
2. 點擊「**新增排程**」（綠色按鈕）
3. 填寫訂單編號、數量等資訊
4. 確認後工單會加入排程佇列

#### 刪除排程
1. 選取要刪除的工單
2. 點擊「**刪除排程**」（紅色按鈕）
3. 確認刪除

> ⚠️ **正在生產的工單**：需確認車速為 0 且未生產量為 0 才能刪除

### 故障排除

| 問題 | 可能原因 | 解決方式 |
|------|----------|----------|
| 無法刪除第一筆 | 工單正在生產中 | 需先完工或車速/數量歸零 |
| 找不到產品 | 產品未建檔 | 使用右側「新增」按鈕建立產品 |
| 排程消失 | 資料未儲存 | 重新載入頁面，檢查網路連線 |

## 3. 生產報表 (Report)

### 功能說明
查詢與匯出生產記錄。

### 報表類型

| 報表 | 說明 |
|------|------|
| **生產明細** | 逐筆查看所有生產記錄與停車明細 |
| **生產日報表** | 依日期彙總當日生產統計 |
| **生產月報表** | 依月份彙總每日生產統計 |
| **停車原因** | 分析停車原因分布與頻率 |

### 操作步驟

#### 查詢報表
1. 點選左側報表類型
2. 設定日期範圍（開始～結束）
3. 選擇班別（可選「全部」）
4. 點擊「確認」載入資料

#### 匯出報表
1. 載入資料後點擊「匯出 Excel」
2. 系統會下載 Excel 檔案

#### 月報表
1. 選擇「生產月報表」
2. 使用年/月下拉選單選擇月份
3. 系統顯示該月每日統計與月度總計

#### 停車原因分析
1. 選擇「停車原因」
2. 設定時間區間
3. 系統顯示各原因的次數與時長
4. **點擊原因行可展開**，查看該原因下每筆訂單的停車細節

### 故障排除

| 問題 | 可能原因 | 解決方式 |
|------|----------|----------|
| 報表無資料 | 日期範圍設定錯誤 | 確認日期是否包含生產記錄 |
| 匯出失敗 | 瀏覽器封鎖下載 | 檢查瀏覽器下載設定 |
| 數據不對 | 資料尚未同步 | 重新整理頁面 |

## 4. 生產分析 (Analysis)

### 功能說明
提供進階的生產效能分析與圖表。

### 分析指標

| 指標 | 說明 |
|------|------|
| **OEE** | 整體設備效率 = 時間稼動率 × 性能稼動率 × 良品率 |
| **時間稼動率** | 實際運轉時間 ÷ 計劃運轉時間 |
| **性能稼動率** | 實際產量 ÷ 理論產量 |
| **良品率** | 良品數量 ÷ 總生產數量 |

### 圖表類型
- OEE 趨勢圖
- 車速趨勢圖
- 不良率分析
- 停車原因分析（圓餅圖）

### 操作步驟
1. 選擇時間區間（今日/本週/本月/自訂）
2. 系統自動計算並顯示圖表
3. 可點擊圖表查看詳細數據

### 故障排除

| 問題 | 可能原因 | 解決方式 |
|------|----------|----------|
| 圖表不顯示 | 無生產資料 | 確認時間區間內有生產記錄 |
| OEE 異常高/低 | 參數設定問題 | 檢查系統設定的計算參數 |

## 5. 保養維修 (Maintenance)

### 功能說明
管理設備保養排程與維修記錄。

### 功能頁籤

| 頁籤 | 說明 |
|------|------|
| **維修清單** | 待處理的維修工單 |
| **零件庫存** | 備品零件的庫存數量 |
| **保養排程** | 定期保養的時程表 |
| **歷史紀錄** | 過去的維修與保養記錄 |
| **照片管理** | 維修相關照片上傳與查看 |

### 操作步驟

#### 回報故障
1. 點擊「故障回報」
2. 選擇故障類型與部位
3. 填寫故障描述
4. 提交後系統會建立維修工單

#### 執行保養
1. 在「保養排程」找到待執行項目
2. 點擊「執行」
3. 完成後填寫執行結果
4. 確認完成

### 故障排除

| 問題 | 可能原因 | 解決方式 |
|------|----------|----------|
| 看不到保養項目 | 權限不足 | 聯繫管理員開通權限 |
| 無法上傳照片 | 檔案太大 | 壓縮照片後重試 |

## 6. 系統設定 (Settings)

### 功能說明
系統參數與進階設定（**通常由管理者操作**）。

### 設定分類

| 分類 | 說明 |
|------|------|
| **一般設定** | 公司名稱、機台名稱等基本資訊 |
| **單位設定** | 公厘/英吋、楞別厚度設定 |
| **機台設定** | 機台極速、部位名稱 |
| **通訊設定** | PLC 連線、ERP 整合設定 |
| **計算公式** | OEE、良率等計算參數 |
| **盒型設定** | 紙箱盒型代碼管理 |
| **報表設定** | 報表產出邏輯參數 |

### 操作權限
- 一般操作員：僅能查看部分設定
- 管理者：可修改所有設定

> ⚠️ **警告**：修改通訊設定可能導致系統連線中斷，請謹慎操作

### 故障排除

| 問題 | 可能原因 | 解決方式 |
|------|----------|----------|
| 無法儲存設定 | 權限不足 | 聯繫管理員 |
| PLC 連線測試失敗 | IP 或埠號設定錯誤 | 確認 PLC 的 IP 位址與通訊埠 |
| ERP 同步失敗 | 網路不通 | 檢查網路連線與防火牆設定 |

## 故障排除總覽

### 連線狀態指示燈

畫面左下角顯示連線狀態：
- 🟢 **綠燈** = 已連線
- 🔴 **紅燈** = 連線中斷

### 常見問題速查

| 問題 | 檢查項目 | 處理方式 |
|------|----------|----------|
| 系統無反應 | 網路連線 | 檢查網路線，嘗試重新整理頁面 |
| PLC 數據不更新 | PLC 連線狀態 | 確認連線燈號，重新連線或重啟服務 |
| 班別選單跳不掉 | 未選取班別 | 點選班別行後再按「選取」 |
| 報表資料不對 | 日期/班別設定 | 確認篩選條件正確 |
| 按鈕無作用 | 系統忙碌中 | 等待數秒後重試 |
